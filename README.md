# Daily Habit Tracker

A beautiful, motivational habit tracker with progress history.

## Features
- Daily habit tracking with checkboxes
- Motivational quotes that refresh
- Calendar view of habit history
- Weekly progress visualization
- Yesterday's summary
- Mobile-responsive design

## How to Use
1. Add habits using the input field
2. Check off completed habits
3. View your progress in the calendar
4. Stay motivated with daily quotes

Data is stored locally in your browser.
